ООП

Task 1.

Реалізуйте клас, що описує геометричну фігуру з властивостями та методами:

  • get-властивість для отримання назви фігури;
  • метод для обчислення площі фігури;
  • метод для обчислення периметра фігури.

Реалізуйте класи-спадкоємці: квадрат, прямокутник, трикутник. Перевизначте методи виведення та обчислень у класах-спадкоємцях. Створіть масив з різними фігурами та виведіть інформацію про кожну з них, включаючи площу та периметр.

Task 2.

Реалізуйте клас, що описує товар з властивостями та методами:

  • назва товару;
  • ціна товару;
  • кількість одиниць товару;
  • метод для виведення інформації про товар в console.log().

Реалізуйте класи-спадкоємці: книга, телефон, холодильник. Перевизначте методи виведення та обчислень у класах-спадкоємцях. Створіть масив з різними товарами та виведіть інформацію про кожен з них.

Task 3.

Реалізуйте клас, що описує новину (заголовок, текст, масив тегів, дата публікації). У класі необхідно реалізувати один метод print, який виводить всю інформацію в console.log(). Зверніть увагу, як виводиться дата:

  • якщо з дати публікації пройшло менше дня, то виводиться «сьогодні»;
  • якщо з дати публікації пройшло менше тижня, то виводиться «N днів тому»
  • в інших випадках – повна дата у форматі «день.місяць.рік».

Реалізуйте класи-спадкоємці: книга, телефон, холодильник. Перевизначте методи виведення та обчислень у класах-спадкоємцях. Створіть масив з різними товарами та виведіть інформацію про кожен з них.

Task 4.

Реалізуйте клас, що описує стрічку новин. Клас має містити:

  • масив новин;
  • get-властивість, яка повертає кількість новин;
  • метод виведення на екран усіх новин;
  • метод додавання новини;
  • метод видалення новини;
  • метод сортування новин за датою (від останніх новин до старих);
  • метод пошуку новин за тегом (повертає масив новин, в яких вказано переданий тег у метод).

Продемонструйте роботу написаних методів.

Task 5.

Реалізуйте клас, що описує автомобіль з властивостями та методами:

  • марка автомобіля;
  • модель автомобіля;
  • рік випуску автомобіля;
  • потужність двигуна;
  • метод для виведення інформації про автомобіль в console.log().

Реалізуйте класи-спадкоємці: легковий автомобіль, вантажний автомобіль, автобус. Перевизначте методи виведення та обчислень у класах-спадкоємцях. Створіть масив з різними автомобілями та виведіть інформацію про кожен з них.
